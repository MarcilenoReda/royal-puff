# 👑 Royal Puff Vapes - Premium E-Liquids Website

A modern, high-performance e-commerce website for Royal Puff premium vape products, built with Next.js 14 and optimized for speed and user experience.

## ✨ Features

- **🎨 Modern UI/UX**: Beautiful, responsive design with smooth animations
- **⚡ Optimized Performance**: Fast loading with Next.js 14 optimization
- **📱 Mobile-First**: Fully responsive design for all devices
- **🛒 Product Showcase**: Interactive product catalog with detailed pages
- **📞 Contact Integration**: WhatsApp, phone, and email integration
- **🎯 SEO Optimized**: Meta tags and structured data for better search visibility

## 🛠️ Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Language**: TypeScript
- **Icons**: React Icons
- **Deployment**: Ready for Vercel/Netlify

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/royal-puff-vapes.git
cd royal-puff-vapes
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

## 📁 Project Structure
